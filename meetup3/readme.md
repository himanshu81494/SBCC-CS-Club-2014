===

**Problem:**

> Given a boolean matrix (double-dimension array) in which each row is sorted, determine which row contains the maximum number of ones (true values). If multiple rows have the same number of ones, output the higher row index.

SAMPLE INPUT:

    0000111
    0011111
    0000001
    0111111
    0001111
    0111111
    0000111

SAMPLE OUTPUT:

    trueMax = 6

Method of bottom-up, left-right searching. Cuts out searching once the highest has been found, removing the need for a full array scan.
